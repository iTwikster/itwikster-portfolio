# iTwikster Portfolio

Single-page portfolio website designed to be sent to employers.

## Run

Open `index.html` directly in a browser or launch the folder in VS Code and use Live Server.

## Update before sending

- Replace the Email card text in `index.html`.
- Replace the LinkedIn and Discord placeholders in `index.html`.
- Update `assets/iTwikster-CV.pdf` if you create a newer CV.
- Add more projects by duplicating a project card in `index.html`.
